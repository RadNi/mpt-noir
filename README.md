<hr>

**MPT Noir** includes tools to prove Ethereum Merkle Patricia Trie (MPT) proofs.



<br>

## Installation

In your `Nargo.toml` file, add the following dependency:

```toml
[dependencies]
ecrecover = { tag = "v0.1.0", git = "https://github.com/radni/mpt-noir" }
```
